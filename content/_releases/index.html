The release notes for the releases of the {{site.data.tenant.name}} platform

NOTE: From 19.51 release we have adopted new release numbering format based on year and week of the release - YY.WW. YY are the last two digits of the year, and WW is the number of the week in that year.

Previously {{site.data.tenant.name}} followed the Semver semantics releases. Given a version number MAJOR.MINOR.PATCH, increment the:

  • MAJOR version when incompatible changes are made in the platform,
  • MINOR version for each monthly release, and
  • PATCH version for bug fix releases.
